A) GLOBAL “EMBED WITH FULLSCREEN” TEMPLATE (REUSE FOR ALL GAMES)
========================================================================

1) Plain HTML wrapper (copy/paste)
------------------------------------------------
<div class="sim-wrap" style="position:relative;width:100%;height:75vh;min-height:520px;border-radius:16px;overflow:hidden;border:1px solid #e5e7eb;">
  <button class="fs-btn" style="position:absolute;z-index:10;top:12px;right:12px;padding:10px 12px;border-radius:10px;border:0;cursor:pointer;background:#111827;color:#fff;">
    Fullscreen
  </button>
  <iframe class="sim-frame"
    src="SIM_URL_HERE"
    style="width:100%;height:100%;border:0;"
    allow="fullscreen"
    loading="lazy"
  ></iframe>
</div>
<script>
  (function(){
    const wrap = document.currentScript.previousElementSibling;
    const btn = wrap.querySelector('.fs-btn');
    btn.addEventListener('click', async () => {
      if (!document.fullscreenElement) { await wrap.requestFullscreen(); btn.textContent='Exit Fullscreen'; }
      else { await document.exitFullscreen(); btn.textContent='Fullscreen'; }
    });
    document.addEventListener('fullscreenchange', () => { if(!document.fullscreenElement) btn.textContent='Fullscreen'; });
  })();
</script>

2) React wrapper component (recommended)
------------------------------------------------
Create <SimEmbed title src /> that:
- renders iframe
- adds fullscreen toggle (requestFullscreen/exitFullscreen)
- supports “open in new tab”
- supports responsive height (vh) and mobile

Acceptance test:
- game is playable
- fullscreen expands ONLY sim container
- ESC exits fullscreen
